# Halt

A macOS menu bar app that reminds you to take breaks.

At a configured interval, Halt shows a full-screen translucent overlay with a message or image of your choice. To dismiss it, you press a key a set number of times — making it hard to ignore without being completely blocking.

## Features

- Configurable reminder interval and repeat behavior
- Custom text or image reminder content
- Configurable dismissal key and required press count
- Pause options and launch at login

## Requirements

- macOS 14 or later

## Building

Open `Halt.xcodeproj` in Xcode and build, or use Swift Package Manager:

```
swift build
```
